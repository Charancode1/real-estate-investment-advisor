# Real Estate Investment Advisor (ML + Streamlit)

This project predicts:
1. Whether a property is a good investment (Classification)
2. Estimated 5-year future price (Regression)

Models Used:
- Random Forest Classifier
- Linear Regression Regressor

Features:
- User-friendly Streamlit UI
- Model explanations
- Visual insights (feature importance, heatmaps)
- MLflow tracking

Deploy:
Hosted via Streamlit Cloud
